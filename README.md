# Personal academic site

Template by [Susung Hong](https://github.com/SusungHong).
Built with [Jekyll](https://jekyllrb.com/) (Ruby).

**Live:** [susunghong.github.io](https://susunghong.github.io)

## Local preview

```bash
bundle install
bundle exec jekyll serve --host 127.0.0.1 --port 4000
```

Open `http://127.0.0.1:4000`.

## Deploy

GitHub Pages is deployed with the workflow in `.github/workflows/deploy.yml` (`bundle exec jekyll build`).
